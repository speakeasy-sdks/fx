# ClientDetailResponseDto2CurrencyAuthorizationType

This field denotes the authorization type of a client. The valid values are SINGLE, DUAL, MULTI, and AUTO_SWEEP.


## Values

| Name         | Value        |
| ------------ | ------------ |
| `SINGLE`     | SINGLE       |
| `DUAL`       | DUAL         |
| `MULTI`      | MULTI        |
| `AUTO_SWEEP` | AUTO_SWEEP   |
